# Scheduler App

Aplikasi generator jadwal kuliah berbasis Python dan Flask.

## Fitur
- Penjadwalan otomatis
- Parameter dinamis
- Export ke Excel

## Cara menjalankan

1. Install dependensi:
pip install -r requirements.txt


2. Jalankan aplikasi:
python app.py


3. Buka di browser:
http://127.0.0.1:5000
